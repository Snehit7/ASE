Class BOOSETriangle
Implements the "tri" command in BOOSE (optional feature). Draws a triangle with the specified base width and height at the current canvas position. Usage: tri <width> <height> Example: tri 100 80 The triangle is centered horizontally at the current position, with the apex at the top.

C#
