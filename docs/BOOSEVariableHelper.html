Class BOOSEVariableHelper
Helper class for type conversion and expression evaluation in BOOSE programs. Provides methods to convert string expressions to int, double, and bool values using the program's evaluator while maintaining precision.
Syntax
C#
    public static class BOOSEVariableHelper

Documentation Preview
Method ToInt
Converts an expression to an integer using the program's evaluator. Supports variable references, arithmetic expressions, and literals.
Syntax
C#
    public static int ToInt(StoredProgram program, string expr)
Parameters
program
The StoredProgram context containing variables.
expr
The expression to evaluate (e.g., "10", "width", "x + 5").
Return Value
The integer result of the expression.
Exceptions
Exception Condition
ArgumentNullException Thrown when program is null.
CommandException Thrown when expression cannot be parsed as integer.
